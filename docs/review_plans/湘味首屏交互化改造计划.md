# 湘味首屏交互化改造计划（含 Pencil MCP 设计对齐）

## Summary

- 将 `Home / Attract Screen`（首页吸引态）改为 `interactive hero carousel`（可交互主视觉轮播）：`Left/Right`（左右键）在首屏内循环切换 5 张推荐卡，右侧 `hero image`（主图）和左侧文案同步切换。
- 保留 `auto-play`（自动轮播）：默认开启；用户手动切卡后暂停；`10s idle timeout`（10 秒空闲超时）后从当前卡继续轮播；轮播间隔固定为 `5s`。
- `Up/Down/Center`（上下/确认键）都直接进入 `Browse`（菜单页）；`Menu key`（菜单键）进入 `Settings`（设置页）；`Browse -> Attract` 的现有 `15s idle timeout` 保持不变。
- 主题令牌 `theme tokens`（主题令牌）和首页 UI 结构落盘前，必须以 `Pencil MCP` 读取 [pencil-new.pen](/Users/PopoY/workingFiles/Projects/POCO/POCO-DISH-VISION/design/pencil-new.pen) 为准，不凭截图肉眼硬编码；代码完成后再用同一设计稿做回看校验。

## Assumptions

- 本轮不修改 `catalog.json` 与 `Browse` 菜单数据语义；首页使用独立的湘味展示数据。
- `Browse` 进入后仍定位到默认分类，不根据首屏当前卡跳分类或跳菜品。
- 首页离开后再次回到 Home，默认重置到第 1 张卡并重新进入自动轮播；“从当前卡恢复轮播”只适用于用户仍停留在同一轮 Home 会话内的 10 秒空闲恢复。
- `Center` 与 `Up/Down` 行为保持一致，均进入 `Browse`；首页不新增“直接开菜品详情”的独立导航分支。
- 所有主题令牌和首页关键 UI 比例，以 [pencil-new.pen](/Users/PopoY/workingFiles/Projects/POCO/POCO-DISH-VISION/design/pencil-new.pen) 的 `Pencil MCP` 读取结果为实现依据。
- 由于 `R3XHK` 中仅提供 1 张显式主图资源，剩余 4 张首屏主图在本轮实现中按用户许可从 `generated-*` 图片池随机选取，后续如设计补齐映射可直接替换资源文件。
