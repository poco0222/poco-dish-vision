# 湘味首屏推荐带调整规格（Hero Recommendation Strip Spec）

## 文档元信息

- 项目名称：POCO Dish Vision
- 页面范围：`湘味首屏`
- 文档类型：Layout Adjustment Spec（布局调整规格）
- 状态：Approved
- 作者：PopoY / Codex
- 创建日期：2026-03-19
- 关联设计文件：`design/pencil-new.pen`

## 1. 背景

当前 [pencil-new.pen](/Users/PopoY/workingFiles/Projects/POCO/POCO-DISH-VISION/design/pencil-new.pen) 中的 `湘味首屏` 底部仅有 1 张推荐菜卡片，信息承载不足，无法满足“首屏直接展示多道推荐菜”的需求。

同时，用户明确要求：

- 将底部推荐菜改为可容纳 5 个推荐菜卡片
- 推荐菜卡片整体向下移动
- 不压缩、不侵占右侧推荐菜主图 `hero image`（主图）空间

## 2. 目标

本次调整的目标是：

1. 将首屏底部单卡推荐区改为 `horizontal strip`（横向单排推荐带）
2. 在一屏内同时展示 5 道推荐菜
3. 保持当前首屏主标题区与右侧主图区的视觉关系不变
4. 通过下移推荐带，强化“首屏主图优先、推荐带辅助点单”的层级

## 3. 设计决策

已确认采用方案：`full-width bottom dock`（整行底部推荐带）。

具体含义：

- 保留右侧 `hero` 主图尺寸与位置
- 删除当前左下角单个 `featureCard`
- 在首屏底部创建一条横向推荐带
- 推荐带内部放置 5 张等宽推荐菜卡片
- 推荐带位置整体下移，靠近首屏底边，但保留安全留白

## 4. 内容规格

推荐菜卡片名称固定为以下 5 道：

1. 茶油炒鸡
2. 青花椒鱼片
3. 钵钵茶油鸭
4. 跳水鱼
5. 水煮活鱼

每张卡片应至少包含：

- 菜名
- 价格或规格信息
- 一句简短推荐理由

如果首屏空间有限，优先保证：

1. 菜名可读性
2. 5 张卡片完整同屏
3. 主图空间不受影响

## 5. 布局要求

### 5.1 主图区域

- `hero` 主图保持现有宽高不变
- 主图不得因推荐带扩容而缩小
- 主图仍然是首屏第一视觉重心

### 5.2 推荐带区域

- 推荐带位于首屏底部区域
- 使用单行横向排列
- 5 张卡片宽度保持统一
- 卡片之间保持一致 `gap`
- 推荐带整体位置较当前单卡更靠下

### 5.3 视觉层级

- 首层级：主标题与主图
- 次层级：底部推荐带
- 推荐带视觉上应像“进店先看这 5 道”的快速点单入口，而不是替代主图的主角区

## 6. 实施边界

本次调整仅处理 `湘味首屏` 的推荐带布局，不改动：

- `湘味菜单`
- `湘味详情`
- 首屏顶部品牌区文案
- 右侧主图素材

## 7. 验收标准

完成后应满足以下结果：

1. 首屏底部可以同时显示 5 张推荐菜卡片
2. 卡片名称与顺序符合已确认内容
3. 推荐带相较当前版本明显下移
4. 右侧主图空间未被压缩
5. 首屏整体没有出现重叠、裁切或拥挤问题
